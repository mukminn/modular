# Contributing

## Prinsip

- PR kecil, jelas, mudah direview
- Hindari overengineering
- Setiap fitur baru usahakan jadi modul terpisah (lihat `plugins/`)

## Setup

- Node.js LTS
- Metamask / WalletConnect

## Struktur

- Contracts: `core-contracts/`
- UI: `frontend-app/`
- API: `backend-api/`

## Standar PR

- Jelaskan tujuan & perubahan
- Sertakan langkah test lokal
- Jangan commit secret (`.env`)
