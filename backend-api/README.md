# backend-api

Backend ringan (opsional) untuk OpenChain Hub.

## Tujuan MVP

- Tidak wajib untuk running MVP (UI bisa langsung baca chain via RPC)
- Endpoint `GET /health` untuk sanity check
- Placeholder untuk indexing/analytics di masa depan

## Local

```bash
npm install
npm run dev
# http://localhost:8787/health
```
